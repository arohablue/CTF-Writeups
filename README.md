# CTF Write-Ups

A collection of capture-the-flag challenge write-ups from various competitions. Each folder contains a detailed walkthrough of the problem, the exploit or solution steps, and any supporting scripts or files.

## 📁 Repository Structure

```
.
├── UMD-2025
    ├── README.md
    ├── challenges
    │   ├── crypto
    │   │   └── brainrot13
    │   ├── misc
    │   │   ├── findtheseeds
    │   │   └── tiktok-ban
    │   ├── osint
    │   │   └── ohio
    │   ├── pwn
    │   │   └── gambling2
    │   └── web
    │       ├── brain-rot
    │       ├── minecraft-movie
    │       └── steve
```

## 🚀 Getting Started


1. **Browse challenges**  
   - Each subfolder under `challenges/` is organized by category.
   - Open its `README.md` for a high-level overview and step-by-step solution.

3. **Run scripts**  
   ```bash
   cd challenges/crypto/brainrot13
   python3 solve.py
   ```
