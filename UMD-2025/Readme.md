# UMD CTF 2025 Write-Ups

Challenge write-ups from UMD CTF 2025.
Please read individual readme file for challenge info and solution.s

## Challenges

- **crypto/**
  - **brainrot13.md**: decrypt a Rot13‐based cipher hidden in a “brainrot” text to recover the flag.

- **misc/**
  - **findtheseeds.md**: reverse‐engineer the PRNG seed used by the challenge to predict outputs and extract the flag.  
  - **tiktok-ban.md**: hijack a DNS TXT record for “tiktok.com” to bypass a legislative filter and reveal the flag.

- **osint/**
  - **ohio.md**: use open‐source intelligence to uncover locations around ohio.

- **pwn/**
  - **gambling2.md**: exploit a race condition in an asynchronous gambling service to manipulate your balance and read the flag.

- **web/**
  - **brain-rot.md**: bypass input filtering on a French‐language site to inject commands and steal the flag.  
  - **minecraft-movie.md**: abuse a backend API in a “Minecraft Movie” guessing game to leak the flag.  
  - **steve.md**: perform a directory traversal on the “Steve the Fish” web app to download the flag file.
